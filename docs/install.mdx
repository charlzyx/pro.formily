# Pro Formily 是什么?

ProFormily 是基于 @formily/antd 而开发的模板组件, 提供了更高级别的抽象与配置简化功能增强, 开箱即用.
可以显著地提升制作 CRUD 页面的效率, 希望能让你少加班.

> 同时, 得益于 API 的相似性, 重用的大部分逻辑代码, 实现了 `@formily/antd` / `@formily/antd-v5` / `arco.formily` 三个基础库的适配.
